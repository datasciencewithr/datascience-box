Getting started

Before the first day of class, please make sure you have an account on GitHub and access to a version of RStudio (either Cloud, or on your computer)

GitHub

You’ll need a GitHub account. Go to GitHub and sign up. Once you have a GitHub account, I can invite you to labs and homework.

RStudio

If you are new to all this, I’d recommend starting with RStudio Cloud. You can always change later.

RStudio Cloud

Just go to RStudio Cloud and click “Get Started For Free”. Follow the steps for Cloud Free. That’s it. We will talk about how you will use this on day 1 of class.

RStudio on your computer

While this is a little more complicated, it will be well worth the effort if you plan on continuing to use RStudio. Do the following to get started:

  • Download and install R. The official instructions are on CRAN
  • Download and install RStudio Desktop Free
  • You will need a local copy of git. Use the instructions here: Using Git with RStudio. This might be tricky, and we can work it out together if you run into trouble.
