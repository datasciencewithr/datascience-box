Computation times to run neural networks of different sizes

Neural network inference computation is the act of runing a neural network on an input to create an output. In our case, the input to the network is an image, and the output is a list of detected objects and where they were found in the image.

We collected data for running neural network inference computations over a variety of variables in order to determine how the variables affected the speed.

Variables of interest:

  • Model: What model and size of model was used

  • gpu: Was computation done on gpu or not (if not then it is on a CPU)

  • network: Were images sent over a network, and if so was it wired or wireless

  • images: number of images in the test

  • time: in seconds to run all the images

  • fps: Frames per second, calculated by dividing images / time.
